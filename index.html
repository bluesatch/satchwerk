"Satch"

Peace Power Blessings. My name is Herbert Brown, but I go by Satch. I spend my work hours teaching others how to fix (and break) things via coding. The Mississippi Coding Academy pays me to do this. When I'm not doing that, I'm probably doing something music related, whether it's DJ'ing, talking about it, or (more conventionally), just listening. Of all the things I do and have done, my greatest passion is being a husband to my wife, Cassandra, and father to my kids, Y'Sonni and Josiah.

What I've learned as an instructor/developer is that I never really "know" languages. I am familiar with them, some more than others, but I am constantly learning each day, and becoming more and more familiar with each block of code I write. Here's a list of the technologies that I have become familiar with over time and continue to learn:

  • HTML/CSS
  • JavaScript
  • Python
  • React
  • PHP
  • SQL
  • Bootstrap
